Explicación de cómo funciona el código

Abrir la cámara: Cuando el usuario pulsa “Iniciar cámara”, el sistema pide permiso para usar la cámara. Si se acepta, el video aparece en pantalla.

Cargar el modelo: Al empezar, se carga un modelo llamado COCO-SSD, que sirve para reconocer objetos como personas, animales o vehículos.

Mostrar lo que detecta: Cuando el modelo reconoce algo, dibuja un rectángulo alrededor del objeto y muestra su nombre junto con el porcentaje de seguridad.

Elegir el más seguro: Si detecta varios objetos, se queda con el que tiene mayor porcentaje de confianza y lo muestra como la mejor detección.

Detectar todo el tiempo: La detección se repite constantemente mientras la cámara está encendida, para que funcione en tiempo real.

Si hay errores: Si ocurre algún problema (por ejemplo, no se puede usar la cámara), se muestra un mensaje de error y se desactiva el botón.
